tags h1 até h6

Os elementos HTML h1 até h6 são utilizados para definir os cabeçalhos de um documento HTML. O h1 é usado para o cabeçalho principal, enquanto o h2 até o h6 são usados para subcabeçalhos 1234. Cada elemento de cabeçalho descreve brevemente o tópico da seção em que ele está 1. Os elementos de cabeçalho são implementados em seis níveis, sendo o h1 o mais importante e o h6 o de menor importância 1.

tag p

Essa tag é utilizada para definir o parágrafo

tag mark

O elemento HTML mark é usado para destacar partes do texto.

tag small

A tag é usada para tornar o texto um tamanho menor

tag i

A tag é usada para apresentar textos em itálico

tag u

A tag u é usada para sublinhar o texto

tag strong

O elemento strong é utilizado em conteúdos que são de "grande importância", incluindo coisas de urgentes (como alertas)

tag ol

A tag HTML ol é usada para criar uma lista ordenada

tag ul

A tag HTML ol é usada para criar uma lista não ordenadas

tag li

O elemento HTML li é usado para representar um item que faz parte de uma lista

tag a

A tag a em HTML é usada para criar um hyperlink, que é usado para linkar de uma página para outra

tag hr

A tag HTML representa uma quebra temática entre elementos de nível de parágrafo

tag sub

A tag sub é usada para definir texto subscrito em HTML

tag sup

A tag sup é usada em HTML para adicionar um texto sobrescrito ao documento

tag blockquote

A tag blockquote é uma tag HTML que indica que o texto incluído é uma longa citação

tag font

A tag font é usada para modificar o tipo, tamanho e cor do texto

tag del

A tag HTML del lindo (ou Elemento HTML de Texto Excluído) representa uma parte do texto que foi excluída
